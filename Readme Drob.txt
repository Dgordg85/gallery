Дробь
=======
changeDegreeFrom10ToAny
Меняет систему счисления числа с десятичной в любую другую (2 - 36). Сначала перевод дробную часть. Затем основную.

fraction
Вспомогательная небольшая функция - выделяет дробную часть

cutNumber
Ищет периодичность в строке. Сначала находим вхождения - потом смотрим каких больше, по длине и по количеству. Потом видоизменяем строку.
